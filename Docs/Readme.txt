INSIMSNIFFER
------------

InSimSniffer is a InSim packet sniffer for the racing simulator Live for Speed.
It connects to the game and displays the contents of each InSim packet that is 
received. This is useful both as a debugging aid and also in learning about how
certain operations are handled by InSim. 

INSTALLATION
------------

InSimSniffer does not have an installer, simply unzip the archive and run the
InSimSniffer.exe executable to start the program.

InSimSniffer requires the .NET Framework 4.0 Client Profile or better to run, 
which you can download through Windows Update or from the following URL:

http://www.microsoft.com/net/download

SOURCE CODE
-----------

The source code and documentation for InSimSniffer can be found on the official 
CodePlex site.

http://insimsniffer.codeplex.com

COMMAND LINE ARGS
-----------------

There are a couple of command line args you can set.

/connect=true - connect to LFS after startup
/minimize=true - start minimized (will minimize to tray if set in InSim > Options)
